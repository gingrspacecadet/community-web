Tailwind CSS Cheatsheet

Layout

What it does: Controls the overall box model and container size of elements. Useful for centering content and defining how the element’s size is calculated.

  • container - centers and sets max width for responsive content
  • box-border - includes padding and border in element’s size
  • box-content - excludes padding and border from size
<div class="container mx-auto"></div>
Display

What it does: Defines how elements behave in layout flow, such as block, inline, flexbox, or grid. Important for controlling layout types and hiding/showing elements.

  • block - element takes full width and starts on a new line
  • inline - element flows inline with text
  • flex - enables flexbox layout for flexible positioning
  • grid - enables grid layout
  • hidden - hides element completely
<div class="flex"></div>
Visibility

What it does: Controls whether elements are visible but take space, or invisible but still occupy space in the layout.

  • visible - element is shown normally
  • invisible - element is hidden visually but still takes up space
Overflow

What it does: Controls how content that exceeds an element’s size behaves — whether it scrolls, hides, or overflows visibly.

  • overflow-auto - adds scrollbars if content overflows
  • overflow-hidden - hides overflow content
  • overflow-scroll - always shows scrollbars
  • overflow-visible - content can overflow visibly
Position

What it does: Controls how elements are positioned relative to normal flow or other elements, enabling fixed, absolute, or sticky placement.

  • static - default normal flow positioning
  • relative - positioned relative to itself, allows offset
  • absolute - positioned relative to nearest positioned ancestor
  • fixed - fixed to viewport
  • sticky - toggles between relative and fixed based on scroll
Top / Right / Bottom / Left

What it does: Moves positioned elements away from edges of their containing block.

Use with relative, absolute, fixed, or sticky elements.

top-0 right-4 bottom-10 left-2
Flexbox and Grid

What it does: Enables flexible and grid-based layouts for complex positioning and alignment.

  • flex - start a flex container
  • inline-flex - flex container that behaves like inline element
  • grid - start a grid container
  • inline-grid - grid container behaving inline

Combine with alignment and spacing utilities for precise control.

Flex Basis

What it does: Sets the initial size of a flex item before growing or shrinking.

Example: basis-1/2 sets flex item to 50% of container.

Flex Direction

What it does: Controls the direction flex items flow.

  • flex-row (default, horizontal)
  • flex-col (vertical)
  • flex-row-reverse, flex-col-reverse
Flex Wrap

What it does: Controls whether flex items wrap to the next line or stay on one line.

  • flex-wrap - allows wrapping
  • flex-nowrap - no wrapping, items stay on one line
  • flex-wrap-reverse - wrap in reverse order
Flex

What it does: Shorthand to set flex-grow, flex-shrink, and flex-basis for flex items.

Examples:

  • flex-1 - grow to fill space
  • flex-auto - grow and shrink as needed
  • flex-initial - default size
  • flex-none - don’t grow or shrink
Flex Grow

What it does: Controls how much a flex item grows relative to others.

Examples:

  • grow (equivalent to grow-1)
  • grow-0 (don’t grow)
Flex Shrink

What it does: Controls how much a flex item shrinks relative to others.

Examples:

  • shrink (shrink-1)
  • shrink-0 (don’t shrink)
Order

What it does: Changes the order of flex/grid items visually.

Examples:

  • order-1, order-2, ...
  • order-first, order-last
Grid Template Columns

What it does: Defines how many columns and their sizes in grid layouts.

Examples:

  • grid-cols-3 (3 equal columns)
  • grid-cols-none
Grid Column Start / End

What it does: Controls which column grid items start and end at.

Examples:

  • col-start-1, col-end-3
Grid Template Rows

What it does: Defines rows and their sizes in grid layouts.

Examples:

  • grid-rows-2
Grid Row Start / End

What it does: Controls which row grid items start and end at.

Examples:

  • row-start-1, row-end-3
Grid Auto Flow

What it does: Controls how auto-placed items flow in the grid.

Examples:

  • grid-flow-row (default)
  • grid-flow-col
  • grid-flow-dense (fills gaps)
Grid Auto Columns

What it does: Controls size of implicitly created columns.

Examples:

  • auto-cols-auto
  • auto-cols-min
  • auto-cols-max
  • auto-cols-fr
Grid Auto Rows

What it does: Controls size of implicitly created rows.

Examples:

  • auto-rows-auto
  • auto-rows-min
  • auto-rows-max
  • auto-rows-fr
Gap

What it does: Adds spacing between grid or flex items.

Examples:

  • gap-4, gap-x-2, gap-y-6
Justify Content

What it does: Aligns flex or grid items horizontally within the container.

Examples:

  • justify-start
  • justify-center
  • justify-between
  • justify-around
  • justify-evenly
Justify Items

What it does: Aligns grid items horizontally within their grid area.

Examples:

  • justify-items-start
  • justify-items-center
  • justify-items-end
  • justify-items-stretch
Justify Self

What it does: Aligns individual grid or flex items horizontally inside their container.

Examples:

  • justify-self-auto
  • justify-self-start
  • justify-self-center
  • justify-self-end
  • justify-self-stretch
Align Content

What it does: Aligns flex or grid lines vertically when there is extra space.

Examples:

  • content-start
  • content-center
  • content-end
  • content-between
  • content-around
  • content-evenly
Align Items

What it does: Aligns flex or grid items vertically within the container.

Examples:

  • items-start
  • items-center
  • items-end
  • items-stretch
Align Self

What it does: Allows individual flex/grid items to override container’s vertical alignment.

Examples:

  • self-auto
  • self-start
  • self-center
  • self-end
  • self-stretch
Place Content

What it does: Shorthand for aligning content both vertically and horizontally inside grid or flex containers.

Examples:

  • place-content-center
  • place-content-start
  • place-content-between
  • place-content-stretch
Place Items

What it does: Shorthand for aligning items both vertically and horizontally inside grid or flex containers.

Examples:

  • place-items-center
  • place-items-start
  • place-items-stretch
Place Self

What it does: Shorthand for aligning a single item both vertically and horizontally within its grid/flex container.

Examples:

  • place-self-auto
  • place-self-center
  • place-self-start
  • place-self-end
  • place-self-stretch
Spacing (Padding, Margin, Space Between)

What it does: Controls spacing inside elements (padding), outside elements (margin), and space between siblings.

Examples:

  • p-4 - padding all sides
  • m-2 - margin all sides
  • space-x-4 - horizontal space between siblings
  • space-y-2 - vertical space between siblings
Sizing (Width, Height, Min/Max)

What it does: Controls element size constraints.

Examples:

  • w-1/2 - width 50%
  • h-64 - fixed height
  • min-w-full - minimum width 100%
  • max-h-screen - maximum height equal to viewport
Typography

What it does: Controls text size, weight, style, spacing, alignment, and transformation.

Examples:

  • text-2xl, font-bold, italic
  • tracking-wide (letter spacing)
  • leading-relaxed (line height)
  • text-center, uppercase
Borders

What it does: Adds borders, border width, style, and rounded corners.

Examples:

  • border, border-2, border-t
  • rounded, rounded-lg, rounded-full
  • border-dashed, border-solid, border-dotted
